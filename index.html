Proton on Linux

What is proton?

Proton is a technology wich is used on Linux to play Windows games.

Imagine Linux and Windows speak 2 different languages and Proton acts like a translator.

There are different versions of Proton and sometimes you need older versions,

but Proton don't works every time sometimes the game is to hard to "tranlate".


How to use Proton?

The good news is that it's very simple:

  • On steam: Open Steam Settings > Compatibility > check “Enable Steam Play for all other titles”.
  • select version:If a game won't launch, right-click the game in your library > Properties > Compatibility, and select a different version of Proton from the list.
