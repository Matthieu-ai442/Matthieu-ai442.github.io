How to Install and Use Proton on Linux

Published on 04/08/2026

Introduction

Proton is a tool developed by Valve that allows you to play Windows games on Linux. This guide covers both installation and usage of Proton on Steam.

Step 1: Install Proton on Linux

  1. Open Steam and go to Settings.
  2. Click on Steam Play.
  3. Check Enable Steam Play for all titles.
  4. Select the Proton version you want to use (e.g., Proton 7.0 or Proton Experimental).

Proton is now enabled for all your games, but you may need to configure it individually for each game.

Step 2: Use Proton to Play Windows Games

Launching a Game

  1. Find a Windows game in your Steam library.
  2. Right-click on the game and select Properties.
  3. In the Compatibility section, check Force the use of a specific Steam Play compatibility tool.
  4. Select the Proton version you want to use for this game.

Troubleshooting

If a game doesn't launch or has issues:

  • Try a different Proton version (e.g., switch from Proton 7.0 to Proton Experimental).
  • Check the game's compatibility on ProtonDB for user reports and fixes.
  • Update your graphics drivers and Steam.

ProtonDB

ProtonDB is a community-driven database where users report how well games work with Proton. You can search for a specific game to see:

  • Compatibility ratings (e.g., Platinum, Gold, Silver, Bronze).
  • User reports with specific fixes or settings.
  • Recommended Proton versions for the game.

Conclusion

Proton is a powerful tool for Linux gamers. With these steps, you should be able to install, configure, and use Proton to play most of your Windows games on Linux without issues!
